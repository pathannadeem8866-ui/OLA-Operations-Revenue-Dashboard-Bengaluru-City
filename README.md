# OLA-Operations-Revenue-Dashboard-Bengaluru-City
This repository contains an end-to-end data analytics project analyzing 100,000 rows (1 lac) of synthesized OLA ride-hailing operational data for Bengaluru city spanning a one-month window. Messy transactional data into operational and revenue-driving insights by leveraging a technical stack of Excel, Microsoft SQL Server (T-SQL), and Power BI.
